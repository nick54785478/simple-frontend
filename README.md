# My Angular Template

<h3>
  說明
</h3>

>* &emsp;&emsp;這是我的簡單前端 Template 範例: 主要用於未來遇到 Angular 的相關模板設計時，可以直接套用。

<h3>技術及相關架構設計</h3>

>* MVVM 架構
>* Angular (框架)
>* PrimeNG (UI Library)
>* PrimeFlex (排版專用，如: RWD 等)


 
<h3>功能介紹:</h3>

>* 簡單表單範例
>* 簡單上傳範例
>* 簡單 PickList 範例
>* 簡單 Inline Editable 範例
>* 簡單雙表範例
>* 動態生成 (根據自定義的json 資料長出個人化頁面)
>* 看板範例 (仿 JIRA 的看板)
>* 圖形範例 (DASHBOARD 統計資訊)
>* 魚骨圖範例 (用來進行魚骨圖分析)
>* FTA 範例 (用來進行 FTA 分析)
>* 簡單問卷 (用於簡單的問券填寫)
